Install a new database

If you're new to Magic, you'd probably like to start with some safe moves towards CRUDifying your backend!

Generate CRUD HTTP endpoints

To start generating CRUD endpoints, in the CRUD generator page, select the newly installed database, and then choose one of the existing tables and CRUDify the table.

You'll see Magic does the magic and automatically generates CRUD HTTP endpoints wrapping the selected database.

SQL studio

Now you have CRUD HTTP endpoints generated for the new database. To see all the tables and test the SQL commands in action, navigate to the SQL studio page.

Start playing around with the SQL commands and see how easy is to go forward from this point. Watch for the shortkeys on top right side of the web page, to use it even more simpler!
