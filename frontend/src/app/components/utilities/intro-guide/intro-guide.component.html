Welcome to Magic

1. Install a plugin database

If you're new to Magic, you'd probably like to start with some safe moves towards generating your CRUD backend. As the first step let's install a new database as a plugin. Go to plugins and install the SQLite Chinook DB plugin.

You can also connect to your existing database from the "Management"/"Configuration" menu.

2. Generate CRUD endpoints

To generate a CRUD backend, select the newly installed database in the CRUD Generator, then click "CRUDify all tables". When you are done creating your backend API you can go to "Management"/"Endpoints" to immediately play with your API.

You can also configure the CRUD process on individual table level by selecting the table you want to configure before generating your CRUD API.

3. Generate a frontend

When you're done generating a CRUD API you can also optionally create an Angular frontend wrapping your API. Go to the CRUD Generator, choose the Frontend tab, give your frontend a name, choose Angular, choose components, and click "Generate".

You can find your newly generated frontend in Hyper IDE in the "/dev/frontend/" folder. Download it locally to play around with it, and/or create an Angular cloudlet from hub if you want to deploy it.

4. SQL Studio

You can also use Magic to manage your database in "SQL Studio" allowing you to immediately execute any SQL towards your database, and also design your database graphically without even knowing much SQL at all.

Start playing around with the SQL commands and see how easy it is to move onwards on your own. Look at the shortkeys in the top right corner of the page for keyboard shortcuts.
