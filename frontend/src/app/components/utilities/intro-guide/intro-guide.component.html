Generate CRUD HTTP endpoints

To generate a CRUD backend, select the newly installed database in the CRUD Generator, then click "CRUDify all tables".

When you are done creating your backend API you can go to "Management"/"Endpoints" to immediately play with your API.

SQL Studio

You can also use Magic to manage your database in "SQL Studio" allowing you to immediately execute any SQL towards your database. Most database plugins contains a list of example SQL commands that you can load using the "Load" button.

Start playing around with the SQL commands and see how easy is to go forward from this point. Watch for the shortkeys on top right side of the web page, to use it even more simpler!
