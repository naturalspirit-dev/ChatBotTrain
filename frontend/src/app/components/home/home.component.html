Magic Cloud

Welcome to Magic Cloud, a virtual cloud system and fifth generation programming platform, making it significantly easier to create modern web apps. To get started, click the question mark in the top right corner of this page, and watch the YouTube video, that will walk you through how to configure Magic initially. Each menu item will show a context sensitive tutorial video as you click the question mark, explaining the basic features for that particular page.

Notice, you will need a valid license for Magic, otherwise Magic will stop working after 47 hours. You can obtain and apply a license in the "Configuration/License" menu item, or buy one directly through my website.
