Magic Cloud

Welcome to Magic Cloud, a virtual cloud system and fifth generation programming platform, making it significantly easier to create modern web apps - to the point where some apps might be created by literally clicking a button. Magic provides the following unique features.

  • Code generator, allowing you to create your web apps by clicking a button
  • Dynamic task scheduler and workflow engine based upon Hyperlambda
  • User administration and audit logging out of the box
  • SQL enterprise management tool, supporting MySQL and SQL Server transparently
  • ++++

With Magic, your productivity will soar and might increase up to 1,000 times for some type of apps. For other types of apps, Magic provides you with services making the end result much more stable and secure. Regardless of what apps you create, with Magic you can host your apps on your own cloud that you can install in Azure, AWS or out of your home on a Raspberry PI if you wish.
