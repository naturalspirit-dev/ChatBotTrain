Magic Dashboard

Magic is an Open Source and Free Software CRUD backend build in ASP.NET Core. Its purpose is to allow you to "magically" do the boring stuff, by using automation techniques, creating 80% of your CRUD endpoints, automatically wrapping your MySQL or MSSQL database - While leaving the fun stuff left for you to implement yourself.

Magic is Open Source and Free Software, but licensed under the Affero GPL license, which implies that you have to create Open Source software yourself to use it for free. If this is not an option for you, you can send an email to thomas@gaiasoul.com and negotiate the purchasing of a proprietary license from me.
