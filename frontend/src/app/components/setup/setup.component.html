Setup wizard

You have to setup your system such that it provides a real authentication scheme. Notice, this will permanently change the state of your installation, so be careful, and make sure you have chosen the correct database type that you wish to use. Make sure you have modified you "magic:auth:secret" configuration setting at this point, before you continue.

When you are done, you can crudify the "magic_auth" database to make the "Users" module work, allowing you to add, edit, and delete users in your installation.
