Keyboard shortcuts

  • F5 - Executes your current Hyperlambda
  • Alt+Space - Autocomplete
  • Tab - Indents selected code
  • Shift+Tab - De-indents selected code
  • Alt-M - Toggles fullscreen
  • Esc - Closes fullscreen
  • Alt+L - Opens up load from snippets collection dialog
  • Alt+S - Saves your content to your snippets collection
  • Alt+A - Saves your content to your snippets collection with a new filename
  • Alt+I - Insert snippet from collection
  • Alt+Q - Opens up this dialog
