Keyboard shortcuts

  • F5 - Executes your current Hyperlambda
  • Tab - Indents selected code
  • Shift+Tab - De-indents selected code
  • F11 - Toggles fullscreen
  • Esc - Closes fullscreen
  • Alt+L - Opens up load file from snippets collection dialog
  • Alt+S - Saves your file to your snippets collection
  • Alt+A - Saves your file to your snippets collection with a new filename
  • Alt+I - Opens up this dialog
