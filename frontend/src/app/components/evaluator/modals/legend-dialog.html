Keyboard shortcuts

  • F5 - Executes your current Hyperlambda
  • CTRL+SPACE / FN+CONTROL+SPACE - Autocomplete
  • TAB - Indents selected code
  • SHIFT+TAB - De-indents selected code
  • ALT-M / OPTION+M - Toggles fullscreen
  • ESC - Closes fullscreen
  • ALT+L / OPTION+L - Opens up load from snippets collection dialog
  • ALT+S / OPTION+S - Saves your content to your snippets collection
  • ALT+A / OPTION+A - Saves your content to your snippets collection with a new filename
  • ALT+P / OPTION+P - Insert snippet from collection
  • ALT+Q / OPTION+Q - Opens up this dialog
